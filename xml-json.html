Xml та Json

XML

У даному розділі було виконано серію практичних завдань із використання XML для опису структурованих даних. Основною метою роботи було закріплення навичок створення XML-документів, використання елементів і атрибутів, роботи з просторами імен (namespace), а також формування вкладених структур даних.

Базова структура XML-документа

Було створено простий XML-документ із атрибутами книги.

XML 
<book xmlns="http://www.contoso.com/book"
    title="Pride And Prejudice"
    price="24.95"
    genre="novel"
    isbn="1-861001-57-8"
    publicationDate="1823-01-28" />

Елементна структура

Було реалізовано вкладені елементи для більш гнучкої структури.

XML 
<book xmlns="http://www.contoso.com/book">
    <title>Pride And Prejudice</title>
    <price>24.95</price>
    <genre>novel</genre>
    <isbn>1-861001-57-8</isbn>
    <publicationDate>1823-01-28</publicationDate>
</book>

XML Serialization Basics

Було розглянуто серіалізацію об’єктів у XML за допомогою System.Xml.Serialization, включаючи налаштування елементів, атрибутів і порядку даних.

C# 
public class BookInfo
{
    public string? Title { get; set; }
    public decimal Price { get; set; }
    public string? Genre { get; set; }
    public string? Isbn { get; set; }
    public string? PublicationDate { get; set; }
}

JSON

У даному розділі було розглянуто основи JSON-серіалізації в .NET з використанням System.Text.Json. Основною метою було навчитися керувати структурою JSON-об’єктів та налаштовувати їх вигляд при серіалізації.

Керування назвами та порядком

Для зміни назв властивостей використано JsonPropertyName, а для порядку — JsonPropertyOrder.

C# 
[JsonPropertyName("employee_name")]
[JsonPropertyOrder(4)]
public string? Name { get; set; }

[JsonPropertyName("employee_id")]
[JsonPropertyOrder(1)]
public int Id { get; set; }

Ігнорування властивостей

Атрибут JsonIgnore використовується для виключення полів із JSON.

C# 
[JsonIgnore]
public string? IgnoreProperty { get; set; }

Серіалізація та десеріалізація

Для перетворення об’єктів у JSON використовується JsonSerializer.

C# 
string json = JsonSerializer.Serialize(employee);
Employee? obj = JsonSerializer.Deserialize(json);
