Виконані завдання

Головним завданням навчальної практики було проходження спеціалізованого курсу для підвищення кваліфікації. Я обрав курс із розробки мовою C#, у межах якого було успішно виконано такі практичні завдання:

1. Багатопоточність у C#

Розроблено програмні рішення з використанням сучасних патернів асинхронного програмування (async/await). Реалізовано потокобезпечну синхронізацію даних, розпаралелювання обчислень та ефективний розподіл ресурсів, що дозволило суттєво підвищити продуктивність застосунків.

Детальніше

2. Інтеграція MySQL та ШІ

Спроектовано та оптимізовано реляційні бази даних, написано складні SQL-запити для швидкого доступу до інформації. Налаштовано безшовну взаємодію БД із застосунками. Крім того, успішно інтегровано та досліджено роботу з нейронними мережами для обробки даних.

Детальніше

3. Робота з XML та JSON

Налаштовано процеси серіалізації та десеріалізації складних структур даних. Реалізовано надійні алгоритми генерації, парсингу та валідації JSON/XML файлів для забезпечення коректного обміну інформацією між різними компонентами системи та зовнішніми API.
